Introduction

TCPIP-Volume1-Practice

This project aims to demomstrate the network knowledge illustrated in the classic book TCP/IP Illustrated, Volume 1: The protocols. Unlike other network tutorial, either only focusing on the theory description, or practical usage, we illustrate them both, theory and practice, in order to make readers fully understand network through scientific method, and construct your own knowledge pyramid of network. The presentation formats in this project includes article, code, and video, which are hosted on github, youtube and patreon.

The project is divided into following sections:

  • LAN of sole device
  • LAN with two devices
  • LAN with three and more devices
  • Network services
  • Router and LAN
  • WAN
