Introduction

TCPIP-Volume1-Practice

  • This project aims to demomstrate the network knowledge illustrated in the classic book TCP/IP Illustrated, Volume 1: The protocols.
  • Unlike other network tutorial, either only focusing on the theory description, or practical usage, we illustrate them both, theory and practice, in order to make readers to understand network through scientfic method, and construct the knowledge pyramid.
  • The presentation formats in this project includes article, code, and video, which are hosted on github, youtube and patreon.
  • We do need your support to continue this project in any forms, including issuing bugs, make suggestions, join the team, and donate money or devices.
